TalentTrack - Full Boilerplate (Angular v20 frontend + Django REST backend w/ JWT)
===============================================================================

This archive contains a starter codebase for TalentTrack with:
- Backend: Django + Django REST Framework + SimpleJWT (JWT auth) configured for PostgreSQL.
- Frontend: Angular v20 source files (Login, Dashboard, AuthService, AuthInterceptor, Routing).
- CSS used as the styling language for components.
- Instructions in README-backend.md and README-frontend.md to initialize full projects locally.

Steps overview:
- Configure backend/.env and PostgreSQL, then run migrations and start Django.
- Scaffold an Angular v20 project using Angular CLI and copy provided src files into it.
